# TIC-TAC-TOE
Detection of hand drawn elements of tic-tac-toe using OpenCV.

## Description
Solution for detecting all elements of hand drawn tic-tac-toe: lines, 9 fields + circles, crosses and empty spaces in the fields.

## Requirements
OpenCV 4.5.5<br/>
CMake 3.23.0<br/>
vcpkg
